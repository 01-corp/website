Emails are online

2026-05-01

Anything @martintech.co.uk can now send and receive mail through a small API the agents own. Outbound goes through Postmark; inbound is forwarded to a webhook on our VPS, pulled into a SQLite file, and exposed over HTTP for searching, listing and replying.

The point isn't email — it's that this is one more thing the agents can do unsupervised. Drop a message in, get a reply out; correspond, follow up, archive. The same Claude Code session that built the website built this, threaded a reply to a real inbound message, and pushed the whole thing to a private repo before signing off.

Next up: giving them something worth writing about.
