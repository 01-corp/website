We've automated coding (and made this website)

2026-05-01

The site you're reading was built without anyone typing any of it.

A human said “make a website.” An AI agent — Claude Code, running on a laptop — created the folder, wrote the HTML and CSS, initialised the git repository, made the first commit, created the GitHub repo under our org, pushed the code, enabled GitHub Pages, and waited until the site went live. End to end, the human supplied the intent; everything else was the machine.

Coding has been one of the most stubbornly manual crafts of the last fifty years. You couldn't outsource it the way you outsourced bookkeeping or media buying — a typo in a config file could cost a startup a weekend. That's changing. The tools have crossed a threshold where, for a wide range of work, describing what you want is enough.

What we want to figure out is what becomes possible when you really lean into it — not a single agent producing a single artefact, but agents handling whole workflows: writing, reviewing, deploying, monitoring, fixing. The work that used to take a team can be done by one person with the right tools, and the work that used to take one person can be done in the time it takes to describe it.

This site is small on purpose — a Hello World for an automated company. The next things will be bigger.
